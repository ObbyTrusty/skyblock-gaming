About the Bot

Skyblock Gaming Utilities is a Hypixel Skyblock discord bot developed from the ground up with tranparency and ease of use in mind. It has a public repository where all the code is laid out transparently and can be read by anyone with an internet connection and some basic python knowledge.

Contributing to the bot in the form of bug fixes or new features is as simple as creating a new pull request in the repository!

Features

Anti-Scammer Features

● /scammer lookup - Check if a player is a known scammer (Powered by Skykings API)

● /scammer guild - Check if a guild has any known scammers (Powered by Skykings API)

● /scammer user - Check if a discord user is a known scammer (BETA)

Discord Server Features

● /server blacklist - Add a user to the scammer database only specific to the current discord server

● /server whitelist - Remove a user to the scammer database only specific to the current discord server

● /server scammerlist - Lists all users added using /server blacklist

● /server config - List the current configuration of the discord server.

Guild Features

● /guild link - Link a Hypixel Guild to the discord server

● /guild unlink - Unlink a Hypixel Guild from the discord server

● /guild setcountvc - Update a voice channel with the count of the number of players in the linked Hypixel Guild

General Features

● /election - View the current Hypixel Skyblock election

● /mayor - View the current Hypixel Skyblock Mayor

● /verify - Verify and link a discord user and a minecraft player

● /unverify - UnVerify and unlink a discord user and a minecraft player
